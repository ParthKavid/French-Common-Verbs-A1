# French-Common-Verbs-A1-Level
 A comprehensive verb practice app for A1 Level.

This app has these 4 modes:

Conjugation drills - verb + pronoun → type conjugation
Mistake review mode - practice only the verbs you got wrong
Verb matching/sorting - group by pattern, match forms
Translation practice - English → French conjugation

All for the present tense.


French Verbs are added in this app : 
⸻

⭐ 1. Essential verbs (MOST IMPORTANT)

Learn these first:
	•	être → to be
	•	avoir → to have
	•	aller → to go
	•	faire → to do/make
	•	dire → to say
	•	pouvoir → can/may
	•	vouloir → to want
	•	devoir → must / have to
	•	prendre → to take
	•	mettre → to put
	•	donner → to give
	•	trouver → to find

⸻
